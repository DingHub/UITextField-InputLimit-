
#Categories of UITextField to limit input
几个UITextField分类，用来限制输入
---
eg: forbid emoji, only money decimal , limit length.<br>
如：禁止表情符， 限制金额格式， 限制长度。<br>

Usage:
---
You can use these categories with code:<br>
可以纯代码使用：<br>

```
textField.isMoney = YES
```

或者简单地在IB里设置：<br>
